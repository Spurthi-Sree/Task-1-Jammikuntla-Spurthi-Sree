🤖 Rule-Based Chatbot

Description

This is a simple Rule-Based Chatbot developed using Python. The chatbot responds to predefined user inputs using if-else conditions. It can handle greetings, answer basic questions, provide help information, and exit when requested by the user.

Features

Handles greetings such as Hi, Hello, and Hey

Responds to common questions

Provides a help menu

Runs continuously until the user exits

Uses simple decision-making logic with if-else statements

Technologies Used

Python

How to Run

Step 1: Save the File

Save the chatbot code as:

rule_based_chatbot.py

Step 2: Open Terminal or Command Prompt

Navigate to the project folder.

Step 3: Run the Program

python rule_based_chatbot.py 

Example

Input: hi

Output: 🤖 ChatBot: Hello! How can I help you today?

Input: bye

Output: 🤖 ChatBot: Goodbye! Have a wonderful day!
