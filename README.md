# 🤖 Rule-Based Chatbot

## 📌 Project Overview

The **Rule-Based Chatbot** is a simple chatbot application developed using Python. The chatbot interacts with users by responding to predefined inputs using rule-based logic and conditional statements.

This project demonstrates the fundamentals of chatbot development, user interaction, and decision-making using Python.

---

## 🎯 Objectives

- Build a simple conversational chatbot.
- Understand rule-based response systems.
- Practice Python conditional statements and loops.
- Improve user interaction handling.
- Demonstrate basic AI chatbot concepts.

---

## 🚀 Features

✅ Handles greetings such as **Hi**, **Hello**, and **Hey**

✅ Responds to common user questions

✅ Provides a help menu

✅ Continuous conversation until the user exits

✅ Uses simple rule-based decision-making

✅ Beginner-friendly implementation

## 🧠 How It Works

1. The user enters a message.
2. The chatbot compares the input with predefined rules.
3. A matching response is selected.
4. The chatbot displays the response.
5. The conversation continues until the user types an exit command.
